# Recipe Finder

A React + Vite app to search for recipes, view details, and watch related YouTube cooking tutorials.  

## Live Demo

[https://stunning-rugelach-6da72b.netlify.app/](https://stunning-rugelach-6da72b.netlify.app/)

## Features

- Search recipes by keyword  
- View recipe details  
- Access YouTube tutorial links  
- Responsive design with Tailwind CSS
